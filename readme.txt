== Description ==

Il plugin permette l'aggiunta di un widget custom con le opzioni di ricerca sul sito Barcheyacht.it
E' possibile modificare il titolo del widget e attivare o disattivare il logo Barcheyacht.it

Le opzioni di ricerca vengono parzialmente caricate direttamente dal sito www.barcheyacht.it
in modo dinamico, tramite Javascript con una chiamata POST.

Al click del pulsante "Ricerca" si apre una nuova finestra
del sito www.barcheyacht.it con i risultati della ricerca.

== Installation ==

1. Carica la cartella `ricerca-barcheyacht` in `/wp-content/plugins/`
2. Attiva il plugin tramite il menu 'Plugin' di WordPress
3. Attiva il widget nella sezione 'Widget' di Wordpress
