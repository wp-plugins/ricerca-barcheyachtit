Il plugin permette l'aggiunta di un widget custom con le opzioni di ricerca del sito Barcheyacht.it

== Description ==

Con il plugin di Barcheyacht Ã¨ possibile fornire ai propri utenti il servizio di ricerca delle vacanze in barca e delle barche a noleggio, tramite un accordo di partnership con barcheyacht si puo richiedere l'attivazione "dell'exchange revenue programme" e iniziare a guadagnare fin da subito: richiedi maggiori informazioni a info[at]barcheyacht.it

E' possibile modificare il titolo del widget.

Le opzioni di ricerca vengono parzialmente caricate direttamente dal sito www.barcheyacht.it
in modo dinamico, tramite Javascript.

Al click del pulsante "Ricerca" si apre una nuova finestra
del sito www.barcheyacht.it con i risultati della ricerca.

== Installation ==

1. Carica la cartella `ricerca-barcheyacht` in `/wp-content/plugins/`
2. Attiva il plugin tramite il menu 'Plugin' di WordPress
3. Attiva il widget nella sezione 'Widget' di Wordpress
